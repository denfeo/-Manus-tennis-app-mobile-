# Tennis Mobile App

A mobile application for booking tennis courts.

## Features
- Court booking
- User profile
- Schedule management

## Tech Stack
- React
- Vite
- Tailwind CSS
- TypeScript

## Development

To run the project locally, follow these steps:

```sh
# 1. Clone the repository
git clone https://github.com/denfeo/-Manus-tennis-app-mobile-.git

# 2. Navigate to the project directory
cd -Manus-tennis-app-mobile-

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

## Deployment

This project is automatically deployed to GitHub Pages upon every push to the `main` branch.
